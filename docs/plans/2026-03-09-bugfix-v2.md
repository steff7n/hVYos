# Bugfix V2 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Fix the confirmed installer, build, and runtime bugs from the bug-hunt and keep a running fix log in `bug_report_v2.md`.

**Architecture:** Work from the outside in: first repair the repo's own test/build entrypoints, then align build configuration, then fix installer control flow and metadata, then resolve remaining runtime/package mismatches. Each bugfix starts with a failing regression test or a configuration regression test before production code changes.

**Tech Stack:** Python 3, pytest, unittest, Bash, kickstart/mock configuration, Markdown docs
