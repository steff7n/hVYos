# linta-keybindings

Searchable keybinding reference overlay for Linta Linux.

## Usage

Press the configured hotkey (default: Mod+Shift+/) to open the overlay.
Or run directly: `linta-keybindings`

## Features

- Auto-detects desktop environment (Niri or KDE Plasma)
- Parses keybindings from:
  - Niri: `~/.config/niri/config.kdl`
  - KDE: `~/.config/kglobalshortcutsrc`
- Searchable and filterable in real-time
- Keyboard-navigable (arrow keys, Esc to close)
- Themed to match Linta's visual identity

## Profiles

Applies to: **[KDE]**, **[Niri]**, **[Combined]**
